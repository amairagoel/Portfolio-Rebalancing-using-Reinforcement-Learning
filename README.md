# Portfolio-Rebalancing-using-Reinforcement-Learning
An algorithmic framework for quantitative portfolio management, optimization, and dynamic rebalancing. This repository provides end-to-end pipelines to construct optimal multi-asset portfolios using Modern Portfolio Theory (MPT) and rigorously backtest adaptive rebalancing strategies against historical market data.


Key Features
Mean-Variance Optimization: Programmatically generates the Efficient Frontier to maximize risk-adjusted returns (Sharpe Ratio) or minimize portfolio variance.

Dynamic Weight Drift Modeling: Simulates real-world cumulative asset drift driven by divergent market returns over time.

Dual-Trigger Rebalancing Engine: Evaluates and executes both periodic (time-based) and threshold-based (tolerance band drift) rebalancing rules.

Performance Analytics: Tracks core quantitative metrics including Maximum Drawdown (MDD), Sortino ratio, turnover rate, and the impact of transaction friction.
